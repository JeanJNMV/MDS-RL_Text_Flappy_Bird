This repository contains the code related to the assignment on [Text Flappy Bird](https://gitlab-research.centralesupelec.fr/stergios.christodoulidis/text-flappy-bird-gym) of the "Reinforcement Learning" course at CentraleSupélec. All the code is contained in the Jupyter notebook `TFB_MCSarsa_MARTINI.ipynb`, which includes the implementation of Monte Carlo Control and SARSA algorithms, as well as the evaluation of their performance and generalization capabilities. The report summarizing the results and analysis is available in `report/report.pdf`.
